Inicio
Bienvenido al formulario de Álvaro Cabrera Fernandez.

Este formulario consta de:

- 2 Preguntas de tipo text area
- 2 Preguntas de tipo select
- 2 Preguntas de tipo select multiple
- 2 Preguntas de tipo checkbox
- 2 Preguntas de tipo radio

Además todas ellas tratan los chistes y momentos amenos que han pasado desde que empezó el curso hasta ahora.

Las preguntas de tipo select multiple (al igual que las de checkbox) pueden tener varias respuestas validas. Para seleccionarlas, mantén pulsada la tecla "Ctrl" y haz clic sobre la opción que deseas seleccionar.

Cada pregunta bien contestada suma 1 punto (en el caso de preguntas con varias respuestas debes dar todas las respuestas validas para recibir el punto entero), las mal contestadas restan siguiendo el mismo criterio.

¡Mucha suerte!
